# Privacy Policy Draft — Black World Map

Last updated: 2026-04-24

Black World Map is a travel tracking application that lets users reveal a personal map by recording their movement.

## Data collected
The app may collect and store the following data on the user's device:
- GPS location points recorded during trips.
- Trip start and end times.
- Distance and discovered map cells.
- App settings such as reveal radius and GPS accuracy preferences.

## Background location
When the user starts a trip, the app may continue recording location in the background so the route can be saved while the phone screen is locked or the app is minimized. A visible notification is shown during background tracking.

## Data storage
The current version stores trip data locally on the user's device. The app does not require an account for local use.

## Data sharing
The app does not sell personal data. GPX exports are created only when the user chooses to export or share a trip.

## User controls
Users can:
- Start or stop tracking manually.
- Review trip history.
- Export GPX files.
- Delete app data by removing the app or clearing app storage.

## Future cloud sync
If cloud sync is added in a later version, this Privacy Policy must be updated before release to explain account data, servers, retention, deletion, and third-party providers.
