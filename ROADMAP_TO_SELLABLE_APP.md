# Roadmap to Sellable App

## Phase 1 — Stable private APK
Goal: install on real Android phones and test daily.

Tasks:
- Generate full Flutter Android/iOS project wrappers.
- Build signed APK/AAB.
- Test permissions.
- Test background tracking.
- Fix crashes and GPS filtering.
- Improve UI polish.

## Phase 2 — Play Store beta
Goal: closed testing with real users.

Tasks:
- Add app icon and splash screen.
- Add permission education screens.
- Add delete trip/data controls.
- Add onboarding.
- Add privacy policy web page.
- Prepare Play Console listing.
- Upload AAB to internal testing.

## Phase 3 — Sellable release
Goal: paid/premium-ready public release.

Tasks:
- Add authentication.
- Add cloud sync.
- Add subscription billing.
- Add crash analytics.
- Add better offline strategy.
- Add achievements and progress.
- Add country/city detection.

## Phase 4 — Strong product
Goal: become a memorable travel app.

Tasks:
- Family maps.
- Trip albums with photos.
- Yearly travel recap.
- Import GPX.
- Web dashboard.
- Social sharing cards.
